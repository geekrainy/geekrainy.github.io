改用linux系统

Posted on Apr 23, 2014

linux 系统作为稳定且免费的操作系统，除了在服务器上的大量使用之外，还吸引了大批程序员，普通用户的追随。用 windows 各种版本用了很久，身在天朝，每次也都是 kms 搞定系统。其实本身自己还是崇尚正版使用的，所以在本周正式改用了稳定的 Linux 发行版 Debian 作为主力系统，一方面想学习 Linux 的服务器构建，另一方面也是为了更原生的编程。

Debian 的安装很容易，但是安装完成后还是多多少少有点小问题，需要进一步调试。

开机后，还没解决中文环境，就发现风扇狂转，温度升高。是双显卡的问题，独显没用只顾发热了，所以参照网上用 bumblebee 解决了双显卡切换问题。安装了文泉驿微米黑后，浏览器的框框终于消失，但字体显示效果和ubuntu还是无法相比，只能慢慢适应这样的字体显示。

安装的时候选择的是 xfce4 桌面环境的版本，对于桌面没有什么特殊要求也不需要配置什么。接下来安装了中文输入法 ibus 平台下的 googlepinyin，智能化程度虽然不能和 win 平台下的相比，不过词库也没太偏僻和匮乏。适应之后打字还是可以很快。默认的语言环境是英文，对于我这个英语渣来说还是稍微费力的，倒是正好可以提高自己的英语水平。

把基本的系统配置好之后，就开始搭建 LAMP 的环境，apt-get 方式安装很方便，几条命令的事情。为了方便管理数据库，安装了 phpmyadmin。安装了一个本地的 typecho 博客，完美运行，也就可以继续折腾 typecho 的模板了。vim 作为编辑器之神，当然不可获缺。奇怪的是在 debian7.4 中用 vi 启动后显示是 vim,但 vim 则无法运行，apt 安装 vim 后则可以了，感觉没有什么变化。使用 vim 是一个耗脑力的工程，只能在以后慢慢熟练了。

用 Linux 的好处好像是可以更专注的进行工作学习吧。至少面对这样一个英文界面不会有娱乐的念头。希望自己可以将这个 linux 系统征服吧。用它做一些更好的作品出来。
