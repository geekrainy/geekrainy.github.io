后记

这个使用姿势是很奇葩的使用姿势，正确的使用姿势应该配合 DNS 查询 API 和 HTTP API 等去做更进一步的集成，在 DNS 查询层级做负载均衡和高可用的配置处理。服务的注册现在是手动注册，结合 Docker 可以使用 Registrator，不适用 Docker 就要写脚本去检测服务并注册了。

因而题为“半自动化”，这个半应该再加个半，为”半半自动化”为宜。w(ﾟДﾟ)w
